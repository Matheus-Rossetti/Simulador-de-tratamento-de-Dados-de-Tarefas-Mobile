# TaskReport Dart
### Simulador de Tratamento de Dados de Tarefas

> Mini-Projeto Avaliativo

---

## Sobre o projeto

O **TaskReport Dart** simula o recebimento de dados de tarefas vindos de uma API. Os dados chegam em formato `List<Map<String, dynamic>>` cheios de inconsistências e o codigo tenta tratá-los.

O sistema transforma esses dados em objetos e gera um relatório no console.

---

## Objetivo

Praticar os principais conceitos de Dart usados no dia a dia de um desenvolvedor Mobile quando tratando dados vindos de uma API.

---

## Como executar

Para rodar o projeto, você precisará do Dart instalado.
Depois é só clonar o repositório e rodar com ```bash dart run main.dart```
